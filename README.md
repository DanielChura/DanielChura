## Sobre mí

Soy un estudiante de Ingeniería de Software enfocado en construir aplicaciones estructuradas, legibles y con altos estándares de calidad. Mi aproximación al desarrollo no se limita al uso automático de frameworks, sino al entendimiento de los principios de diseño de software y la ingeniería detrás de ellos.

### Lo que me importa al construir software:

*   **Código limpio y buenas prácticas:** Mantengo buenas prácticas en el control de versiones y escribo código modular con responsabilidades claras, buscando que sea fácil de leer y mantener.
*   **Separación de responsabilidades:** Priorizo que cada módulo tenga un propósito claro y que cualquier miembro del equipo pueda entender el código sin fricción.
*   **Visión de extremo a extremo:** Entiendo el flujo completo diseño de APIs, persistencia de datos y entrega de interfaces funcionales, bajo los criterios que se definan.
